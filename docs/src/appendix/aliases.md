# Command Aliases

Ironclad provides a few short aliases:

- `ic rm` -> `ic remove`
- `ic sh` -> `ic show`
- `ic ls` -> `ic list`
- `ic r` -> `ic resolve`
- `ic i` -> `ic inspect`
- `ic d` -> `ic diff`
- `ic c` -> `ic check`
- `ic up` -> `ic apply`
- `ic op ls` -> `ic op list`
- `ic op sh` -> `ic op show`
